# Codex

Control the Codex app from Raycast. This extension opens key Codex views so you can start a new thread or jump straight to settings, skills, and automations without leaving your keyboard.

## Commands

1. `New Thread`: Opens a new thread in the last project you had open in Codex.
2. `Open Settings`: Opens the Codex settings menu.
3. `Open Skills`: Opens the skills menu.
4. `Open Automations`: Opens the automations menu.

## Requirements

- macOS
- Codex app installed
